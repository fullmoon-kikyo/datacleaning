# datacleaning

S4 数据清洗与模具过程组件分析脚本。

## 脚本

- `extract_mold_versions.py`: 读取 `1510-过程组件分析结果-20260421.xlsx` 的 `分析结果` 工作表，解析 `子件号` 并生成模具版本分析列。
- `process_mold_data.py`、`clean_mold_data_20260414.py`、`clean_bom_transform.py`: 既有数据清洗脚本。

## 运行环境

```powershell
pip install pandas openpyxl tqdm
```

## 模具版本分析

运行：

```powershell
python extract_mold_versions.py
```

输入文件：

- `1510-过程组件分析结果-20260421.xlsx`
- 工作表：`分析结果`
- 必要列：`子件号`

输出文件：

- `1510-过程组件分析结果-20260421【处理后】YYYYMMDD-HHMMSS.xlsx`
- 仅导出处理后的 `分析结果` 工作表。

新增列：

- `成套模具号`
- `初始模具号`
- `变更履历`
- `当前版本`
- `版本数量`
- `A版` 到 `F版`
- 检测到 A-F 以外版本时，自动追加对应版本列，例如 `G版`、`P版`、`S版`、`Y版`

处理规则：

- `子件号` 含 `/` 时，取 `/` 前内容作为 `成套模具号`；不含 `/` 时，整串作为 `成套模具号`。
- `成套模具号` 末尾为字母时，去掉末尾字母得到 `初始模具号`，末尾字母作为版本。
- 按 `初始模具号` 分组汇总版本，`变更履历` 格式为 `【SY】`，`当前版本` 格式为 `-Y-`，`版本数量` 为该组版本个数。
- 脚本使用 UTF-8 输出中文提示。
